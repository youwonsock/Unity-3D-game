\doxysubsection{상세한 설명}
Represents RFC4330 SNTP packet used for communication to and from a network time server. 

Most applications should just use the \mbox{\hyperlink{class_flying_worm_console3_1_1_lite_net_lib_1_1_utils_1_1_ntp_packet_a47b6edd3b8fec9efb4003c300e748afc}{Ntp\+Packet.\+Correction\+Offset}} property. 

The same data structure represents both request and reply packets. Request and reply differ in which properties are set and to what values. 

The only real property is \mbox{\hyperlink{class_flying_worm_console3_1_1_lite_net_lib_1_1_utils_1_1_ntp_packet_ae74b153f215ec9464c01b478434050f6}{Ntp\+Packet.\+Bytes}}. All other properties read from and write to the underlying byte array with the exception of \mbox{\hyperlink{class_flying_worm_console3_1_1_lite_net_lib_1_1_utils_1_1_ntp_packet_af6bab33467fcd3d991e277a406f7d989}{Ntp\+Packet.\+Destination\+Timestamp}}, which is not part of the packet on network and it is instead set locally after receiving the packet. 

Copied from \href{https://guerrillantp.machinezoo.com/}{\texttt{ Guerrilla\+Ntp project}} with permission from Robert Vazan (@robertvazan) under MIT license, see \href{https://github.com/RevenantX/LiteNetLib/pull/236}{\texttt{ https\+://github.\+com/\+Revenant\+X/\+Lite\+Net\+Lib/pull/236}}

Gets the offset that should be added to local time to synchronize it with server time. 

Time difference between server and client. It should be added to local time to get server time. It is calculated from timestamps in the packet as {\ttfamily 0.\+5 $\ast$ ((t1 -\/ t0) -\/ (t3 -\/ t2))} where t0 is \mbox{\hyperlink{class_flying_worm_console3_1_1_lite_net_lib_1_1_utils_1_1_ntp_packet_ae5feb244befa8720954e523b3fd3c77e}{Ntp\+Packet.\+Origin\+Timestamp}}, t1 is \mbox{\hyperlink{class_flying_worm_console3_1_1_lite_net_lib_1_1_utils_1_1_ntp_packet_aa373ab879c13fd6149d81b20f143c337}{Ntp\+Packet.\+Receive\+Timestamp}}, t2 is \mbox{\hyperlink{class_flying_worm_console3_1_1_lite_net_lib_1_1_utils_1_1_ntp_packet_a31be651514ce9f209563d8c59a92e193}{Ntp\+Packet.\+Transmit\+Timestamp}}, and t3 is \mbox{\hyperlink{class_flying_worm_console3_1_1_lite_net_lib_1_1_utils_1_1_ntp_packet_af6bab33467fcd3d991e277a406f7d989}{Ntp\+Packet.\+Destination\+Timestamp}}.
